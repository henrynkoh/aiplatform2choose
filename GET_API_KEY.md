## Step-by-Step Instructions

### Step 1: Visit the Link
Click or copy this link: **https://console.groq.com/keys**

### Step 2: Sign Up (if you don't have an account)
- Click "Sign Up" or "Get Started"
- You can sign up with:
  - Google account (fastest)
  - Email address
- It's completely FREE - no credit card required

### Step 3: Create API Key
- Once logged in, you'll see the API Keys page
- Click **"Create API Key"** button
- Give it a name (e.g., "BuildFast Advisor")
- Click **"Submit"** or **"Create"**

### Step 4: Copy Your Key
- Your API key will appear (looks like: `gsk_xxxxxxxxxxxxx...`)
- **IMPORTANT:** Copy it immediately - you won't see it again!
- It starts with `gsk_` followed by random letters/numbers

### Step 5: Add to Your Project
1. Open the file `.env.local` in this project
2. Find this line: `GROQ_API_KEY=your_groq_api_key_here`
3. Replace `your_groq_api_key_here` with your actual key
4. Save the file
5. The dev server will automatically reload!
